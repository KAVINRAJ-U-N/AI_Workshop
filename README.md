# Personal Portfolio Website

A responsive, static personal portfolio website built with HTML5 and CSS3.

## Features
- **Responsive Design**: Mobile-first approach, compatible with Mobile, Tablet, and Desktop.
- **Pages**: Home, About, Skills, Projects, Contact.
- **Theme**: Professional Blue & Gray color scheme.
- **PWA Ready**: Includes manifest and offline support.
- **No Dependencies**: Pure HTML/CSS only.

## Structure
- `index.html`: Home page.
- `about.html`: About section with bio and education.
- `skills.html`: Technical capabilities.
- `projects.html`: Portfolio showcase.
- `contact.html`: Contact information and form.
- `css/`: Stylesheets.
- `assets/`: Images and icons.

## How to Run
Simply open `index.html` in any modern web browser.
